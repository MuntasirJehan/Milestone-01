Git কী?

Git একটি ওপেন-সোর্স ভার্সন কন্ট্রোল সিস্টেম (VCS), যা ডেভেলপারদের একটি প্রজেক্টের সোর্স কোড ট্র্যাক করতে, সংস্করণ সংরক্ষণ করতে এবং একাধিক ডেভেলপারের সাথে সহযোগিতায় কাজ করতে সাহায্য করে। এটি মূলত সফটওয়্যার ডেভেলপমেন্টের জন্য ব্যবহৃত হয়, তবে অন্যান্য ফাইল বা ডকুমেন্ট পরিচালনায়ও কার্যকর। Git তৈরির পেছনে মূল উদ্দেশ্য হলো পরিবর্তন ট্র্যাকিং, যেখানে ডেভেলপাররা ফাইলের পরিবর্তনের ইতিহাস সংরক্ষণ করতে পারে এবং প্রয়োজনে আগের সংস্করণে ফিরে যেতে পারে।
